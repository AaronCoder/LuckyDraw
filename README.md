# Lucky Draw

一个基于 Three.js 的幸运抽奖系统，支持 3D 动画效果和独立应用打包。

## 功能
- 3D 动画效果
- 支持独立运行的可执行文件
- 简单易用的幸运抽奖系统

## 使用方法

### 运行可执行文件
1. 下载打包好的可执行文件（例如 `luckydraw.exe`）。
2. 在终端中运行以下命令：
   ```bash
   ./lucky-draw.exe
   ```

3. 打开浏览器，访问 `http://localhost:3000`，即可使用幸运抽奖系统。

### 注意事项
- 确保运行环境中已安装 Node.js（如果需要动态加载其他模块）。
- 打包后的文件无需额外的依赖，直接运行即可。
